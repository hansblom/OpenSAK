// Features

Everything you need.
Nothing you don't.

Built by geocachers, for geocachers. OpenSAK handles the heavy lifting so you can focus on the hunt.

📥

GPX & Pocket Query Import

Import GPX files and Pocket Query ZIP archives with full Groundspeak field support including logs, attributes, and waypoints.

🔍

Advanced Filter Engine

18 filter types with AND/OR nesting. Save and reuse filter profiles. Filter by D/T, type, size, attributes, date, distance, and more.

🗺️

Interactive OSM Map

Colour-coded cache pins with clustering on OpenStreetMap. Corrected coordinates for mystery caches shown with an orange overlay.

📡

Garmin GPS Export

Send your filtered cache selection directly to a connected Garmin device via USB — ready for offline field use.

🧮

Coordinate Tools

Built-in converter, projection calculator, checksum tool, midpoint finder, and distance/bearing calculator.

🌍

Multilingual

Full interface in English, Danish, and French — with more languages on the way. Community translations welcome.

🗄️

Multiple Databases

Manage separate databases for different regions or trip planning. Switch between them instantly.

🔒

100% Local & Private

All data stays on your own machine. No cloud, no accounts, no tracking. Your geocaching data is yours.
